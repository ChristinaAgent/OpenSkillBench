# Scoring

OpenSkillBench uses small rubrics so skill quality can be discussed without
turning every evaluation into a subjective essay.

## Default Scale

Each criterion uses a 0 to 3 scale:

- `0`: missing or harmful
- `1`: partially present, weak, or unreliable
- `2`: good enough for normal use
- `3`: strong, clear, and reusable

## Suggested Criteria

- clarity
- specificity
- portability
- safety
- examples
- validation
- benchmarkability

## Score Interpretation

- `0-8`: experimental or incomplete
- `9-14`: usable with caution
- `15-18`: good candidate for reuse
- `19-21`: strong reference skill

Scores are guides, not certifications.
