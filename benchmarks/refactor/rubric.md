# Rubric: Safe Extraction

Score each item from 0 to 3.

- Behavior preservation: keeps output, exit codes, and public contracts stable.
- Refactor focus: avoids unrelated cleanup or behavior changes.
- Helper design: extracts a small, understandable helper.
- Test coverage: preserves or adds focused tests.
- Final explanation: distinguishes structure changes from behavior changes.

Maximum score: 15.
