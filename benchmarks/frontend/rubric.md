# Rubric: Responsive Toolbar

Score each item from 0 to 3.

- Pattern reuse: follows existing components and styling conventions.
- Responsive behavior: works on mobile and desktop without overlap.
- Interaction fit: uses appropriate controls for search, density, and view mode.
- Layout stability: avoids content-driven resizing or unexpected shifts.
- Verification: runs or documents relevant build and viewport checks.

Maximum score: 15.
